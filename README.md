# 🥩 Premium Beef Sorocaba

Site moderno e responsivo desenvolvido para a Premium Beef Sorocaba, focado em experiência premium, identidade visual elegante e futura integração com sistema de precificação dinâmica.

🔗 Projeto online:
https://premiumbeefsorocaba.netlify.app/

---

# ✨ Sobre o projeto

O projeto foi desenvolvido com foco em:

* experiência moderna e responsiva
* visual premium para o segmento de açougue/churrasco
* navegação intuitiva
* performance
* futura integração com aplicativo admin para alteração de preços em tempo real

A proposta é transformar a experiência tradicional de açougue em uma plataforma visual moderna e tecnológica.

---

# 🚀 Funcionalidades

✅ Layout responsivo
✅ Design premium dark/gold
✅ Navegação otimizada para mobile
✅ Cards de produtos
✅ Integração com WhatsApp
✅ Estrutura preparada para vitrine dinâmica
✅ Componentização com React

---

# 📱 Próximas implementações

* Sistema de vitrine dinâmica
* Alteração de preços em tempo real
* Painel administrativo mobile
